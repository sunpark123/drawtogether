# 멀티 공유 화이트보드 ( DrawToGether )

다 같이 함께 또는 혼자 그림을 그리는 멀티 공유 화이트 보드 웹입니다.
실시간으로 여러 사용자가 동시에 그리기, 지우기, 색상/브러시 변경, 채팅 등을 할 수 있습니다.
